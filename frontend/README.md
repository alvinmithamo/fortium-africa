# Fortium Africa Digital Foundation

## Project Overview

Fortium Africa's digital foundation website showcasing engineering solutions for energy, water, and civil infrastructure projects across Africa.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd fortium-africa-s-digital-foundation

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

## Development

To work on this project locally:

1. Clone this repository
2. Install dependencies with `npm i`
3. Start development server with `npm run dev`
4. Make your changes and commit them to the repository

## Building

To create a production build:

```sh
npm run build
```

## Technologies

This project is built with:

- **Vite** - Fast build tool and dev server
- **React** - UI framework
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn-ui** - High-quality UI components
- **React Router** - Client-side routing
- **Lucide React** - Icon library
